---

# Why Does the Math Class Exist?

Before learning methods like:

```csharp
Math.Max()
Math.Min()
Math.Sqrt()
Math.Abs()
Math.Round()
```

ask a deeper question:

> Why doesn't C# just rely on normal arithmetic operators like `+`, `-`, `*`, and `/`?

Because real software performs far more complex calculations.

Examples:

```text
Find the larger discount
Calculate distance
Round currency values
Generate game physics
Compute scientific formulas
Determine absolute differences
```

Arithmetic operators solve basic mathematics.

The Math class solves advanced mathematical problems.

---

# What Is the Math Class?

The Math class is a built-in utility class provided by .NET that contains methods and constants for performing mathematical operations.

Namespace:

```csharp
System
```

Example:

```csharp
int largest = Math.Max(10, 20);
```

Output:

```text
20
```

# Why Is Math a Class?

Why not create keywords?

Instead of:

```csharp
sqrt 25
```

C# uses:

```csharp
Math.Sqrt(25);
```

Because:

- Groups related functionality together
- Improves discoverability
- Keeps the language smaller
- Enables code organization

Because Math is a static class.

# Math.Max()

---

# What Is It?

Returns the larger of two values.

Example:

```csharp
Console.WriteLine(Math.Max(5, 10));
```

Output:

```text
10
```

---

# Why Does It Exist?

Without it:

```csharp
if (a > b)
{
    max = a;
}
else
{
    max = b;
}
```

Math.Max simplifies this.

---

# Internal Working

Mental model:

```text
Compare first value
↓
Compare second value
↓
Return larger value
```

---

# Real-World Usage

```csharp
highestScore = Math.Max(player1, player2);
```

---

# Common Mistake

Thinking:

```text
Math.Max finds the maximum in a list.
```

Reality:

It compares two values at a time.

---

# Hidden Behavior: Banker's Rounding

One of the most surprising behaviors.

Example:

```csharp
Console.WriteLine(Math.Round(2.5));
```

Output:

```text
2
```

Example:

```csharp
Console.WriteLine(Math.Round(3.5));
```

Output:

```text
4
```

---

# Why?

By default, .NET uses:

```text
MidpointRounding.ToEven
```

Also called:

```text
Banker's Rounding
```

---

# Mental Model

Exactly halfway values round to the nearest even number.

```text
2.5 → 2
3.5 → 4
4.5 → 4
5.5 → 6
```

---

# Why Does This Exist?

Reduces rounding bias across large datasets.

Useful in financial and statistical systems.

---

# Custom Rounding

Example:

```csharp
Math.Round(2.5, MidpointRounding.AwayFromZero);
```

Output:

```text
3
```


```text

```


Because:

```text

Floor   → Toward negative infinity
```

Not:

```text

```

# Interview Perspective

## What is the Math class?

Answer:

The Math class is a static utility class in .NET that provides methods and constants for mathematical operations.

---

## Why is Math static?

Answer:

Because mathematical operations belong to the class itself rather than individual objects.

---

## Difference between Ceiling and Floor?

Answer:

Ceiling rounds upward toward positive infinity, while Floor rounds downward toward negative infinity.

---

## What does Math.Abs() do?

Answer:

Returns the absolute value, representing the distance from zero.

---

## What is Banker's Rounding?

Answer:

The default rounding behavior in .NET where midpoint values round to the nearest even number.

Example:

```text
2.5 → 2
3.5 → 4
```

---

## What happens when Math.Sqrt() receives a negative number?

Answer:

It returns NaN because real-number square roots of negative values do not exist.

---

# Best Practices Followed by Experienced Developers

- Use Math methods instead of reinventing mathematical logic.
- Be aware of Banker's Rounding.
- Use MidpointRounding when business rules require it.
- Validate inputs before calling Sqrt().
- Remember that Pow() returns double.
- Prefer readability over clever formulas.
- Test edge cases involving negatives and decimals.
- Understand the business meaning behind rounding decisions.
