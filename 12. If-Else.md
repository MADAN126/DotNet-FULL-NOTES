```csharp

    // code
```

> Why do programs need `if` statements?

Imagine software that executes every line exactly the same way every time.

Example:

A login system:

```text

â†“
```

# What Is an If Statement?

An `if` statement allows a program to execute a block of code only when a condition evaluates to `true`.

Example:

```csharp
if (true)
{
    Console.WriteLine("Executed");
}
```

Output:

```text
Executed
```

---

# Why Is It Needed?

Without decision-making:

Programs could:

```text
Calculate
Store data
Display output
```

But they could NOT:

```text
Validate users
Authorize actions
Handle errors
Apply business rules
Respond to situations
```




Because once a condition succeeds:

# Short-Circuit Evaluation

Example:

```csharp
if(name != null && name.Length > 0)
{
}
```

Why?

If:

```text
name == null
```

Then:

```text
name.Length
```

never executes.

Prevents:

```text
NullReferenceException
```

# Interview Perspective

## What is an if statement?

Answer:

An `if` statement executes a block of code only when its condition evaluates to `true`.

---

## What is the purpose of else?

Answer:

It specifies the code to execute when the `if` condition evaluates to `false`.

---

## What is else if used for?

Answer:

It allows checking multiple conditions sequentially until one evaluates to `true`.

---

## Does an else-if chain execute multiple branches?

Answer:

No. Once a condition succeeds, the remaining branches are skipped.

---

## What is a nested if statement?

Answer:

An `if` statement placed inside another `if` statement.

---

## What is the ternary operator?

Answer:

A shorthand form of `if...else` used for simple conditional expressions.

---

# Best Practices Followed by Experienced Developers

- Use meaningful conditions.
- Order conditions from most specific to least specific.
- Always use curly braces.
- Avoid excessive nesting.
- Prefer logical operators when appropriate.
- Use ternary operators only for simple decisions.
- Leverage short-circuit evaluation safely.
- Focus on expressing business intent clearly.
